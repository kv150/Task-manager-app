# Task-manager-app
# Interactive Task Manager App

A modern, responsive, and feature-rich **Task Manager Application** built using pure Frontend technologies. This project demonstrates clean coding practices, efficient state management, and seamless DOM manipulation.

---

 # Features

- **Full CRUD Operations**: Users can **Create**, **Read**, **Update** (edit task text/toggle completion), and **Delete** tasks.
- **State Management with Local Storage**: All tasks and user preferences persist even after refreshing the page or closing the browser.
- **Dynamic Filtering**: Easily filter tasks by **All**, **Completed**, and **Pending** status.
- **Dark & Light Mode Toggle**: Smooth theme switching with user preference saved automatically in Local Storage.
- **Fully Responsive**: Optimized for both mobile and desktop screens.

---

 # Tech Stack Used

- **HTML5**: For semantic layout and structure.
- **CSS3**: For custom styling, layout (Flexbox), and Dark/Light theme variables.
- **JavaScript (ES6+)**: For dynamic DOM manipulation, event listeners, and Local Storage integration.
